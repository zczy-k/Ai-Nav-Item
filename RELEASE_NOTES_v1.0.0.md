# Browser Extension v1.0.0 Release Notes

## 🎉 首个正式版本发布！

Con-Nav-Item 浏览器扩展 v1.0.0 正式发布，为您的个人导航站提供完整的浏览器集成体验。

## ✨ 核心功能

### 🏠 新标签页导航
- **导航站模式** - 新标签页直接显示您的个人导航站
- **快速访问面板** - 基于浏览历史的智能书签面板
- **自定义导航站地址** - 支持配置任意导航站 URL

### 🚀 快速添加网站
- **右键菜单添加** - 在任意网页右键，选择分类快速添加到导航站
- **浮动快捷按钮** - 网页右下角浮动按钮，一键添加当前页面
- **智能标签生成** - 自动根据网站域名、标题生成标签
- **编辑后添加** - 添加前可编辑书签标题
- **实时同步** - 添加后导航站页面立即刷新显示新卡片

### 📚 书签管理器
- **本地书签管理** - 查看、搜索、编辑浏览器书签
- **智能排序** - 按使用频率、最近访问、字母顺序排序
- **标签系统** - 为书签添加自定义标签，支持标签筛选
- **笔记功能** - 为书签添加备注笔记
- **批量操作** - 批量添加到导航站、批量删除、批量打标签
- **热门书签** - 自动更新热门书签文件夹
- **重复检测** - 检测并清理重复书签

### ☁️ 云端同步
- **书签云备份** - 将浏览器书签备份到导航站服务器
- **多设备同步** - 在不同设备间同步书签数据
- **自动同步** - 支持定时自动同步

## 🌐 浏览器支持

### ✅ 完全支持
- **Chrome** 88+
- **Microsoft Edge** 88+
- **Brave** 1.20+
- **Opera** 74+

### ✅ Firefox 支持
- **Firefox** 127+
- 使用 Manifest V3 和 browser_specific_settings

## 📦 安装方法

### Chrome/Edge/Brave
1. 下载 `nav-extension-v1.0.0.zip`
2. 解压到本地文件夹
3. 打开 `chrome://extensions/`（Edge 用 `edge://extensions/`）
4. 开启"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压后的文件夹

### Firefox
1. 下载 `nav-extension-v1.0.0.zip`
2. 打开 `about:debugging#/runtime/this-firefox`
3. 点击"临时载入附加组件"
4. 选择 zip 文件

## ⚙️ 配置指南

### 首次设置
1. 点击扩展图标，进入设置页面
2. 输入您的导航站地址（如 `https://your-domain.com`）
3. 输入管理员密码进行验证
4. 选择新标签页模式（导航站/快速访问面板）

### 书签管理器配置
1. 在扩展设置中开启书签管理功能
2. 配置云备份（可选）
3. 设置自动同步间隔

## 🔧 使用技巧

### 快速添加网站
- **方法一**：右键网页 → "添加到导航站" → 选择分类
- **方法二**：点击页面右下角浮动按钮
- **方法三**：在书签管理器中批量添加

### 新标签页优化
- 设置为导航站模式，每次打开新标签页都是您的导航站
- 使用快速访问面板查看常用和最近访问的书签

### 书签管理
- 使用标签系统为书签分类
- 添加笔记记录重要信息
- 定期清理重复书签

## 🔒 隐私与安全

- **本地存储** - 所有配置信息存储在浏览器本地
- **加密传输** - 与导航站服务器的通信使用 HTTPS
- **权限最小化** - 仅请求必要的浏览器权限
- **开源透明** - 完整源代码开放，可审查安全性

## 🐛 已知问题

- Firefox 中某些网站的右键菜单可能不显示（浏览器限制）
- 部分网站的浮动按钮可能被 CSP 策略阻止
